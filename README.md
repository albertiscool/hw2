## 📋 作業概述

本作業在經典的 **Cliff Walking（懸崖行走）** 環境中，實作並比較兩種 Temporal Difference 強化學習演算法：

| 演算法 | 類型 | 更新規則 |
|--------|------|----------|
| **Q-learning** | Off-policy（離策略） | Q(s,a) ← Q(s,a) + α[r + γ·**max** Q(s',a') − Q(s,a)] |
| **SARSA** | On-policy（同策略） | Q(s,a) ← Q(s,a) + α[r + γ·Q(s',**a'**) − Q(s,a)] |

## 📊 結果分析

### 策略行為差異

- **Q-learning**（Off-policy）：傾向學習**沿懸崖邊行走的最短路徑**（理論最優），但訓練中因探索而頻繁掉落懸崖
- **SARSA**（On-policy）：傾向學習**遠離懸崖的安全路徑**（繞上方走），訓練過程更穩定

### 收斂與穩定性

- Q-learning 可能收斂較快，但訓練獎勵波動大（掉崖導致的 -100 尖峰）
- SARSA 訓練過程更平穩，後期平均獎勵通常更高

### 適用場景

| 場景 | 推薦演算法 |
|------|-----------|
| 犯錯代價低、追求最優策略 | Q-learning |
| 犯錯代價高、需要安全行為 | SARSA |
| 機器人控制、自動駕駛 | SARSA（On-policy） |
| 離線學習、經驗回放 | Q-learning（Off-policy） |
